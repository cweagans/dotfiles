## Get Safari Tabs Alfred Workflow

![Get Safari Tabs](src/screenshot.png)

### What does it do?

Copy tabs from Safari in various formats:

- Copy current tab or all
- Choose which of the pages to copy if copying all
- Copy as:
    - Markdown
    - Markdown list
    - Titles only 
    - Links only
    - Titles and links

Keyword is `sf`
